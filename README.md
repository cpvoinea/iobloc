# iobloc
## v1.0 (released)
* 6 games: tetris, runner, helicopter, breakout, invaders, snake
* single-panel box drawing with colored full blocks
* level selection
* volatile highscores
* rudimentary logging

* 7th game: sokoban **done**

* output agnostic: background color, window size, encoding **partial**
